- hwan: Kotlin 에서 길이가 서로 같은 collection 을 돌면서 무언가를 해야 할 때는 `zip` 을 쓰면 편하다.
+
[source,kotlin]
----
val list1 = listOf(1,2,3)
val list2 = listOf(4,5,6)
list1.zip(list2).forEach { (item1, item2) ->
    println("$item1 : $item2")
}
----
